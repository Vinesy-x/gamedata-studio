# GameData Studio — 功能版本规划（精简版）

> **工具定位**：纯离线的 Excel Add-in，不依赖网络/后端/账号  
> **策划装了就能用，无需任何配置**

---

## 总览

| 版本 | 定位 | 核心交付 |
|------|------|---------|
| v1.0 | 替代 VBA | 导出功能完整迁移到 Add-in，与 VBA 结果 100% 一致 |
| v2.0 | 控制面板 | 所有管控操作迁移到侧边栏，策划不再需要编辑元数据表 |
| v3.0 | 质量保障 | 校验引擎 + 版本预览 + 一键全流程，减少配置错误 |

---

## v1.0 — 替代 VBA

**目标**：Add-in 能做 VBA 宏做的所有事情，导出结果 100% 一致。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 导出引擎 | 完整移植 VBA 的版本筛选、差异对比、文件写入逻辑 |
| 2 | Git 集成 | 拉取最新 + 提交推送，替代 WScript.Shell 命令行 |
| 3 | 网页协作导出 | OneDrive 同步方式，替代 VBA 的 Worksheet_Calculate 检测 |
| 4 | 基础侧边栏 | 导出按钮、Git 按钮、进度显示、结果列表、错误日志 |

**验收标准**：对实际数据表执行导出，与 VBA 导出文件逐单元格对比，100% 一致。

---

## v2.0 — 控制面板

**目标**：策划不再需要打开「表格输出」「配置设置表」「表名对照」这三张元数据表。

| # | 功能 | 说明 |
|---|------|------|
| 1 | Studio 控制面板 | 替代「表格输出」：版本选择器（记忆上次值）、版本号、序列号、操作人、导出范围勾选、运行状态、结果列表、错误日志 |
| 2 | 配置可视化管理 | 替代「配置设置表」手动编辑：版本列表、线路列表、人员代码、Git 模板、功能开关，全部改为侧边栏表单操作 |
| 3 | 数据表注册管理 | 替代「表名对照」手动编辑：自动扫描未注册的数据表（含 #配置区域# 标记）、一键注册、批量管理 |
| 4 | 新表创建向导 | 填写表名+字段列表 → 自动生成完整工作表（version_r/roads/#配置区域#/字段定义/中文描述）+ 自动注册。支持撤销 |

**向后兼容**：所有操作双写（侧边栏 + 工作表同步），VBA 仍可并行使用。稳定后「表格输出」隐藏。

---

## v3.0 — 质量保障

**目标**：减少配置错误，提升导出信心。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 数据校验引擎 | 手动触发。格式校验（版本区间语法、数据类型、数组分隔符）+ 逻辑校验（版本覆盖完整性、同Key版本顺序、Roads一致性、必填字段） |
| 2 | 版本预览与高亮 | 选择任意版本+版本号 → 即时显示筛选结果汇总 → 在 Excel 中高亮（绿色保留/灰色排除/黄色被覆盖） |
| 3 | 一键全流程 | 校验 → 导出 → Git 推送，一个按钮串联，任何一步失败自动中断 |

---

## 已确认的产品决策

| 决策项 | 确认结果 |
|-------|---------|
| 版本号记忆 | 存在「表格输出」工作表，所有人共享 |
| 校验触发时机 | 仅手动点击「校验」按钮 |
| 网页协作方式 | OneDrive 同步（onChanged 事件监听） |
| 身份识别（网页协作） | 沿用原有机制（操作人名称匹配） |
| 新表向导撤销 | 支持一键撤销（删除工作表+取消注册） |
| 「表格输出」处置 | v2.0 稳定后隐藏 |
| 网络依赖 | 无。所有功能完全离线 |
| AI 功能 | 不做 |
| 代理服务 | 不做 |
| 规则引擎 | 不做 |
| 用户登录体系 | 不做 |
